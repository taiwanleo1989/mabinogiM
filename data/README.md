# 瑪奇 Mobile 資料庫（Mabinogi Mobile Data）

新手友善攻略網站的資料來源與內容整理。全球資料蒐集，韓國一手資料為主。

## 檔案結構
```
mabinogi-mobile/
├── site/                     ← 可部署的 App（網站根目錄；未來給 Capacitor 打包用）
│   ├── index.html            ← 攻略網站（fetch 讀取 game-data.json）
│   ├── game-data.json        ← ⭐ 單一資料源（Single Source of Truth）
│   ├── manifest.webmanifest  ← PWA 設定（可「加入主畫面」當手機 APP）
│   └── icon.svg              ← App 圖示
└── data/
    ├── README.md             ← 本檔（說明）
    └── sources.md            ← 全球資料來源清單
```

## ⭐ 單一資料源（重要）
攻略內容只存在一份：**`site/game-data.json`**。網站（未來的 APP 也一樣）都用 fetch 讀取它。
**更新攻略時，只改這個檔案**，網站與 APP 會自動一致——這是為了「兩平台資料一致」而設計的架構，
也讓之後用 Capacitor 把同一份程式碼包成手機 App 時無痛接軌。

## 當成手機 APP 安裝
本站是 PWA（漸進式網頁應用）。在手機瀏覽器開啟後，選單選「加入主畫面」，
即可像原生 APP 一樣以全螢幕啟動、有自己的圖示。

## 使用方式（需透過伺服器開啟）
因為資料改用 fetch 讀取，**不能直接雙擊 `index.html`**（瀏覽器 CORS 會擋本機檔案讀取）。
請在 `site/` 資料夾啟動一個簡單伺服器再開，例如：
```
cd site
python -m http.server 8080
# 然後瀏覽器開 http://localhost:8080
```
部署到任何靜態主機（GitHub Pages、Netlify、Vercel…）後，直接開網址即可，也能在手機「加入主畫面」當 App。
若不小心雙擊開啟，頁面會顯示紅色提示教你怎麼正確開。
